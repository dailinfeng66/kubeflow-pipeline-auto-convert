方法调用节点有两种类型，一种是xxx.xxx()的方式。一种是xxx()的方式，这两种方式对应的call节点是不一样的。

xxx.xxx()对应的call节点如下：

```python
Call(func=Attribute(
            value=Name(
              id='params',
              ctx=Load()),
            attr='add',
            ctx=Load()),
          args=[Constant(
            value='sk2',
            kind=None)],
          keywords=[keyword(
            arg='value',
            value=Constant(
              value=1.0,
              kind=None))])
```

xxx()对应的call方式如下所示：

```python
Call(func=Name(
            id='load_data',
            ctx=Load()),
          args=[],
          keywords=[]))
```
